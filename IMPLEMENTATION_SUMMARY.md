
### Step 2: Create GitHub Account & Repository
1. Go to https://github.com/signup
2. Create new repository: `growx-digital-agency-site`
3. Make it PUBLIC

### Step 3: Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit - agency website with admin panel"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/growx-digital-agency-site.git
git push -u origin main
```

### Step 4: Deploy to Netlify
1. Go to https://app.netlify.com
2. Sign up with GitHub
3. Click "Import an existing project"
4. Select your GitHub repo
5. Click "Deploy site"

**Your site is now live!** 🎉

### Step 5: Enable Admin Panel
1. In Netlify: **Site Settings** → **Identity** → **Enable Identity**
2. **Git Gateway** → **Enable Git Gateway**
3. **Invite users** → Enter your email
4. Check email, set password
5. Visit `yourdomain.netlify.app/admin`

### Step 6: Connect Custom Domain (Optional)
1. In Netlify: **Domain Settings** → **Add custom domain**
2. In GoDaddy: Update nameservers
3. Wait 15-30 minutes for propagation
